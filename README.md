Auto Form Filler is a lightweight Windows tool that teaches itself the position of form fields on your screen (via image recognition) and auto-fills them for you — no coding required.

## Features
- **Teach once, run anywhere** — capture a label once during setup; the app relocates it on screen every run, even if you scroll.
- **Anchor points** for forms with repeated/similar labels (e.g. MARC cataloging fields).
- **Auto-scroll search** — automatically scrolls to find fields on long pages.
- **Checklist** — pick exactly which fields to fill each run.
- **Light / Dark / Sepia themes** and adjustable font size.

## Getting Started
1. Download and extract `AutoFormFiller.zip`
2. Run `AutoFormFiller.exe`
3. Click **"+ เพิ่มฟิลด์ใหม่ (Setup)"** to teach the app your form's fields
4. Fill in values and hit **Run All**

**Requirements:** Windows 10/11 (64-bit)

⚠️ This is an early release — please back up important data and test on a non-critical form first.
