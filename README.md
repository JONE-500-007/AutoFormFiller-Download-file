Auto Form Filler is a lightweight Windows tool that teaches itself the position of form fields on your screen (via image recognition) and auto-fills them for you — no coding required.

## Features
- **Teach once, run anywhere** — capture a label once during setup; the app relocates it on screen every run, even if you scroll.
- **Anchor points** for forms with repeated/similar labels (e.g. MARC cataloging fields), plus automatic ambiguity detection that skips a field instead of risking a wrong click.
- **Auto-scroll search** — automatically scrolls to find fields on long pages, with configurable step size and retry rounds.
- **Two fill strategies** — go through fields in order, or fill whichever one appears first while scrolling.
- **Reorderable checklist** — pick exactly which fields to fill each run, and reorder them with one click.
- **Run All / Stop** with global keyboard shortcuts (default F3 / F4) that work even when another window is focused — fully customizable in Settings.
- **Light / Dark / Sepia themes**, adjustable font size, and a resizable window that adapts to any screen size.
- **English and Thai UI** — switch languages anytime in Settings.

## Download Options
Choose the one that fits you:

- **`AutoFormFiller.zip`** — portable, no installation. Just extract and run `AutoFormFiller.exe` directly. Good if you don't want to install anything, or want to carry it on a USB drive.
- **`AutoFormFiller_Setup_1.1.0.exe`** — full installer. Installs to Program Files, adds Start Menu/Desktop shortcuts, and can be removed later via Windows Settings > Apps like any normal program.

## Getting Started
1. Download and extract `AutoFormFiller.zip` **or** run `AutoFormFiller_Setup_1.1.0.exe` and follow the installer
2. Run `AutoFormFiller.exe`
3. Click **"+ Add Field (Setup)"** to teach the app your form's fields
4. Fill in values and hit **Run All** (or press F3)

**Requirements:** Windows 10/11 (64-bit)

⚠️ This is an early release — please back up important data and test on a non-critical form first.
